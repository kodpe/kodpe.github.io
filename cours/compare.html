### WINDOWS / LINUX --- #### Pourquoi Linux est privilégié dans les écoles et l’apprentissage ? Dans l’enseignement supérieur et les métiers techniques, **Linux est aujourd’hui plus utilisé que Windows**. Ce choix est principalement pédagogique et professionnel. Linux est privilégié car : - il est **open source** et **gratuit**, ce qui permet une utilisation libre et sans contrainte de licence - il est **entièrement personnalisable**, ce qui facilite la compréhension du fonctionnement interne d’un système - il est **léger et performant**, même sur des machines modestes - il met en avant le **terminal et les concepts fondamentaux** (processus, fichiers, permissions, réseau) - l’environnement utilisé à l’école est **très proche de celui utilisé en entreprise** Windows reste cependant dominant pour le **grand public**, notamment sur les ordinateurs personnels. --- #### Pourquoi Linux domine dans le monde professionnel de l’informatique ? Linux est le système de référence dans l’informatique professionnelle pour plusieurs raisons : - Environ **95 % des serveurs** fonctionnent sous Linux (sites web, cloud, bases de données, microservices) - Il est **stable**, conçu pour fonctionner longtemps sans redémarrage - Il est **scriptable et automatisable**, ce qui est essentiel en : - DevOps - cloud - cybersécurité - data science - Il respecte davantage la **vie privée** : - pas de télémétrie intrusive par défaut - contrôle total des services actifs - Il est **moins exposé aux virus courants** : - modèle de permissions strict - séparation claire des droits utilisateur / administrateur - moins ciblé par les malwares grand public --- #### Pourquoi Windows est si populaire auprès du grand public ? Windows reste le système le plus utilisé sur les postes personnels pour des raisons pratiques : - Il est **préinstallé** sur la majorité des ordinateurs vendus - Il offre une **interface graphique intuitive**, adaptée aux débutants - Il est **très bien supporté pour les jeux vidéo** (pilotes, DirectX) - Il dispose de la **plus grande base de logiciels grand public** : - bureautique - logiciels professionnels spécialisés - jeux - Il est largement utilisé dans : - les bureaux - les administrations - les entreprises non informatiques --- #### Qu’est-ce que WSL (Windows Subsystem for Linux) ? **WSL** est une fonctionnalité de Windows permettant d’exécuter un environnement Linux **directement à l’intérieur de Windows**. Avec WSL : - Windows reste le système principal - Linux fonctionne comme un sous-système - les **vraies commandes Linux** sont disponibles (`ls`, `cd`, `grep`, `bash`…) - aucun double démarrage n’est nécessaire WSL est très utilisé dans l’enseignement et le développement car il permet : - d’apprendre Linux sur un poste Windows - d’avoir un environnement proche de la production - de combiner logiciels Windows et outils Linux --- #### Résumé - **Linux** est privilégié pour l’apprentissage et les métiers techniques - **Windows** domine chez le grand public - **WSL** permet de réunir les deux mondes sur une même machine
