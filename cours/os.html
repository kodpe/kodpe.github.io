OS, noyau et shell

Comprendre Linux, Windows et WSL — Cours introductif

1. Le système d’exploitation (OS)

Un système d’exploitation (Operating System) est le logiciel fondamental qui permet à un ordinateur de fonctionner.

Il agit comme un intermédiaire entre :

  • le matériel (processeur, mémoire, disque, clavier…)
  • les programmes
  • l’utilisateur

Sans système d’exploitation, le matériel est inutilisable.

Exemples de systèmes d’exploitation

  • Linux
  • Windows
  • macOS
  • Android
  • iOS

2. Le noyau (Kernel)

Le noyau est le cœur du système d’exploitation.

C’est le seul programme qui communique directement avec le matériel.

Rôles du noyau

  • gérer le processeur (qui travaille, quand)
  • gérer la mémoire
  • gérer les périphériques
  • empêcher les programmes de se nuire entre eux
L’utilisateur n’interagit jamais directement avec le noyau.

3. Le shell

Le shell est une interface textuelle permettant à l’utilisateur de donner des ordres au système.

C’est un programme qui :

  • lit les commandes tapées au clavier
  • les interprète
  • demande au noyau de les exécuter

Exemples de shells

  • bash
  • zsh
  • fish

Exemple : 

ls

Le shell comprend la commande, le noyau accède au disque, et le résultat est affiché à l’écran.

4. Relation entre OS, noyau et shell

Utilisateur
    ↑
Shell
    ↑
Noyau
    ↑
Matériel
  • Le noyau contrôle le matériel
  • Le shell permet de dialoguer avec le système
  • Le système d’exploitation est l’ensemble

5. Linux

À l’origine, Linux désigne uniquement un noyau.

Un système Linux complet (souvent appelé GNU/Linux) contient :

  • le noyau Linux
  • un shell (bash)
  • des commandes de base (ls, cp, mv…)
  • éventuellement une interface graphique

6. Windows

Windows est également un système d’exploitation complet, mais son architecture est différente de Linux.

Windows possède :

  • son propre noyau (noyau Windows NT)
  • une interface graphique dominante
  • un shell en ligne de commande

Shells sous Windows

  • cmd.exe (ancien)
  • PowerShell (moderne)
Les commandes Windows sont différentes de celles de Linux.

7. WSL (Windows Subsystem for Linux)

WSL permet d’exécuter un environnement Linux directement à l’intérieur de Windows.

Avec WSL :

  • Windows reste le système principal
  • Linux fonctionne comme un sous-système
  • on utilise les vraies commandes Linux

Exemple de commandes sous WSL : 

ls
cd
pwd
WSL est très utilisé pour l’apprentissage, le développement et les cours d’informatique.

8. Résumé

  • OS : fait fonctionner l’ordinateur
  • Noyau : gère le matériel
  • Shell : interface de commande
  • Linux : noyau + outils
  • Windows : OS différent avec son propre noyau
  • WSL : Linux à l’intérieur de Windows
