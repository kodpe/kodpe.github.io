Introduction au terminal Linux

Cours ultra débutant – Comprendre et utiliser les commandes de base

1. Qu’est-ce qu’un terminal ?

Le terminal est une interface textuelle qui permet de communiquer directement avec le système d’exploitation. Contrairement aux interfaces graphiques (fenêtres, icônes, souris), le terminal fonctionne uniquement avec du texte.

Sous Linux, le terminal permet :

  • d’explorer les fichiers
  • de créer, déplacer ou supprimer des dossiers
  • d’exécuter des programmes
  • d’administrer le système
Le terminal peut sembler intimidant au début, mais il est précis, rapide et extrêmement puissant.

2. Ouvrir un terminal

Selon votre environnement Linux :

  • Raccourci courant : Ctrl + Alt + T
  • Ou via le menu : Applications → Terminal

Une fois ouvert, vous voyez une ligne semblable à : 

utilisateur@ordinateur:~$

Cette ligne s’appelle le prompt. Elle indique que le terminal attend une commande.

3. Où suis-je ?

La commande pwd

Pour savoir dans quel dossier vous vous trouvez : 

pwd

Cette commande affiche le chemin complet du dossier courant.

4. Lister les fichiers

La commande ls

Pour afficher le contenu du dossier courant : 

ls

Pour afficher plus de détails : 

ls -l

5. Se déplacer dans les dossiers

La commande cd

Aller dans un dossier : 

cd Documents

Revenir au dossier précédent : 

cd ..

Revenir au dossier personnel : 

cd

6. Créer et supprimer

Créer un dossier

mkdir cours_linux

Supprimer un fichier

rm fichier.txt

Supprimer un dossier (attention)

rm -r dossier
Sous Linux, une suppression est définitive. Il n’y a pas de corbeille par défaut.

7. Lire un fichier texte

La commande cat

cat fichier.txt

Elle affiche le contenu du fichier dans le terminal.

8. Commandes essentielles à retenir

  • pwd – afficher le dossier courant
  • ls – lister les fichiers
  • cd – changer de dossier
  • mkdir – créer un dossier
  • rm – supprimer
  • cp – copier
  • mv – déplacer ou renommer
  • cat – lire un fichier
  • man – lire le manuel d’une commande
